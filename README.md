# מערכת תורים למספרת כלבים

פרויקט לניהול תורים למספרת כלבים.

## טכנולוגיות

### צד שרת
- .NET 8 Web API
- Entity Framework Core
- SQL Server
- JWT Authentication
- BCrypt

### צד לקוח
- React
- TypeScript
- Material UI
- Axios

## יכולות המערכת

- הרשמת משתמש חדש
- התחברות למערכת
- אימות משתמשים באמצעות JWT
- צפייה ברשימת תורים
- הוספת תור חדש
- עריכת תור
- מחיקת תור
- סינון לפי שם לקוח
- סינון לפי טווח תאריכים
- הצגת פרטי תור בחלון קופץ
- הנחה של 10% ללקוחות בעלי יותר מ־3 תורים קודמים

## אבטחת מידע

- סיסמאות נשמרות בצורה מוצפנת באמצעות BCrypt
- אימות משתמשים באמצעות JWT
- משתמש יכול לערוך ולמחוק רק תורים השייכים לו
- לא ניתן למחוק תור של אותו היום

## מסד נתונים

המערכת עושה שימוש ב:

### Stored Procedure

`sp_CreateAppointment`

לצורך יצירת תור חדש.

### View

`vw_AppointmentsDetailsSwish`

לצורך שליפת נתוני התורים.

## הרצת הפרויקט

### צד שרת

```bash
dotnet restore
dotnet run
```

### צד לקוח

```bash
npm install
npm run dev
```
